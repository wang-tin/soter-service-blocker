## 功能
强力拦截 SoterService，阻止任何应用（包括 com.chunqiunativecheck）调用它

## ✅ 特点
- **100% 安全**，不会影响开机
- 仅使用安全的系统属性设置
- 不修改 SELinux
- 不使用 iptables
- 不使用 mount bind
- 不冻结/删除任何应用

## 2 种安全拦截方法
1. **系统属性**：禁用 soter 和 zygote 相关系统属性
2. **拦截标记**：创建特殊标记文件

## 常见问题

### Q1: v1.8 和之前版本有什么区别？
**v1.8 移除了所有可能影响开机的代码**：
- ❌ 移除了 SELinux 宽容模式（可能导致不开机）
- ❌ 移除了 iptables 规则（可能导致网络异常）
- ❌ 移除了 mount bind（可能导致文件系统异常）
- ✅ 只保留安全的系统属性设置

### Q2: 如何确认模块正常工作？
```bash
cat /data/local/tmp/soter_blocker_log.txt
```

### Q3: 卸载后会影响手机吗？
**不会**。v1.8 只修改了临时文件和系统属性，卸载后完全恢复原状。
